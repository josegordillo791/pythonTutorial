#  Sintaxis:
Para indexar la sintaxis de Python se utiliza el espacio(tabular),no es necesario utilizar punto y coma (;)
```
print('Ejemplo de sintaxis basica de Python')

if 1>2:
  print('1 es menor que 2')
else:
  print('2 es mayor a 1')
```
